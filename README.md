# 🔐 Password Strength Checker

> A beginner-friendly Python project that analyzes the strength of a password and generates secure random passwords.

---

## 📖 About

Password Strength Checker is a simple Python-based application that helps users evaluate the strength of their passwords based on common security rules. It also provides suggestions for improving weak passwords and includes a built-in random password generator.

This project was built to practice Python fundamentals such as functions, modules, loops, conditionals, imports, and project organization.

---

## ✨ Features

- 🔍 Checks if the password has at least 8 characters
- 🔠 Detects uppercase letters
- 🔡 Detects lowercase letters
- 🔢 Detects numbers
- 🔐 Detects special characters
- 📊 Calculates a password score
- 💪 Classifies password strength (Weak / Medium / Strong)
- 💡 Provides suggestions to improve weak passwords
- 🎲 Generates a secure random 8-character password
